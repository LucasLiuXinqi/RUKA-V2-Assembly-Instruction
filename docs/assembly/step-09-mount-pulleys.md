### Instructions

#### 1. Insert the string

Feed the string through the pulley\
![](https://ruka.gitbook.io/instructions/~gitbook/image?url=https%3A%2F%2F971518572-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F4cbJtpXfHVXgNHLpW7Xq%252Fuploads%252Ft9uGJMs9lNP2bhYXZEET%252FScreenshot%25202025-05-31%2520at%252011.31.41%25E2%2580%25AFPM.png%3Falt%3Dmedia%26token%3Dda41392f-e466-461a-8d0c-5d0c3fc0039f\&width=768\&dpr=3\&quality=100\&sign=bf8ff4bd\&sv=2)

#### 2. Loosely attach the pulley to the motor

Attach the pulley with two screws loosely such that the string can still be pulled\
![](https://ruka.gitbook.io/instructions/~gitbook/image?url=https%3A%2F%2F971518572-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F4cbJtpXfHVXgNHLpW7Xq%252Fuploads%252FUbjxrFcogolZI1K8eXoe%252FScreenshot%25202025-05-31%2520at%252011.31.53%25E2%2580%25AFPM.png%3Falt%3Dmedia%26token%3D38c14756-83f9-4c31-9d96-7f0ec686f532\&width=768\&dpr=3\&quality=100\&sign=322e24b3\&sv=2)

#### 3. Tie the knot

Tie the knot in the string as close to the pulley as possible, then pull the string through.

![](https://ruka.gitbook.io/instructions/~gitbook/image?url=https%3A%2F%2F971518572-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F4cbJtpXfHVXgNHLpW7Xq%252Fuploads%252Fz1ohG7VyGwDzN9jlUjN9%252FScreenshot%25202025-05-31%2520at%252011.32.04%25E2%2580%25AFPM.png%3Falt%3Dmedia%26token%3D1c804502-76f3-4e52-bc00-c1e33a5ddfb4\&width=768\&dpr=3\&quality=100\&sign=7bbfa21f\&sv=2)![](https://ruka.gitbook.io/instructions/~gitbook/image?url=https%3A%2F%2F971518572-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F4cbJtpXfHVXgNHLpW7Xq%252Fuploads%252FN59JXvmqmShS87GWFFnC%252FScreenshot%25202025-05-31%2520at%252011.32.19%25E2%2580%25AFPM.png%3Falt%3Dmedia%26token%3Dab47c1c3-6321-47c4-94cd-eb3fa17e1246\&width=768\&dpr=3\&quality=100\&sign=8a0e754c\&sv=2)

#### 4. Tighten the screws all the way

Repeat this for all motors.
