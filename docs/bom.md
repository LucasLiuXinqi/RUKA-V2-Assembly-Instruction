This document lists all components required to build the RUKA-V2 robotic hand.

|-----------|----------|----------|-------|

| Finger Medial | 5 | PLA/ABS | One per finger |
| Finger Distal | 5 | PLA/ABS | One per finger |
| Wrist Adapter | 1 | PLA/ABS | Interface to mounting system |
