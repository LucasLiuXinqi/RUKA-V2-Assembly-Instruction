# Home

This documentation provides step-by-step instructions for assembling, wiring, and calibrating **RUKA-V2**, a tendon-driven robotic hand.

## Build Flow

Follow the sections in order:

1. Bill of Materials and Tools
2. 3D Printed Parts
3. Mechanical Assembly
4. Electronics and Power-On Test
5. Calibration
6. Troubleshooting

> ⚠️ **Safety Notice**\
> This system contains moving mechanisms and powered actuators. Always power the system with current limits enabled and keep fingers clear during testing.
