# RUKA-V2 Assembly Instructions

This documentation provides step-by-step instructions for assembling, wiring, and calibrating **RUKA-V2**, a tendon-driven robotic hand.

The guide is intended for researchers and students who wish to reproduce or study the mechanical and electromechanical design of the system.

## Build Flow
Follow the sections in order:

1. Introduction, Bill of Materials, and Tools
2. Mechanical Assembly
3. Electronics and Power-On Test
4. Calibration and Motion Verification
5. Troubleshooting

> ⚠️ **Safety Notice**  
> This system contains moving mechanisms and powered actuators. Always power the system with current limits enabled and keep fingers clear during testing.
