# Tools

The following tools are required for assembly:

## Hand Tools
- Hex drivers (M2 / M3)
- Tweezers
- Small needle-nose pliers
- Flush cutters

## Assembly Aids
- Threadlocker (medium strength)
- Calipers (recommended)
- Soldering iron (for electronics)

## Safety
- Eye protection
- ESD precautions when handling electronics
