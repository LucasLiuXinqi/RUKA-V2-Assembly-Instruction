# Tools Required

This page lists all tools needed for assembling the RUKA-V2 robotic hand.

## Essential Tools

### Hand Tools

- **Screwdriver Set**
  - Phillips head (#0, #1, #2)
  - Flathead (small and medium)
- **Hex Key Set**
  - Metric sizes (1.5mm, 2mm, 2.5mm, 3mm)
- **Pliers**
  - Needle-nose pliers
  - Wire cutters
  - Wire strippers
- **Tweezers**
  - For handling small components

### Power Tools

- **Soldering Iron**
  - Temperature controlled (recommended)
  - Minimum 30W
- **3D Printer** (for printing parts)
  - Build volume: 200mm x 200mm x 200mm minimum
- **Drill** (optional)
  - For modifications or mounting

## Measurement Tools

- **Calipers**
  - Digital or analog
  - For precise measurements
- **Multimeter**
  - For electrical testing and debugging

## Assembly Aids

- **Workbench or Stable Work Surface**
- **Good Lighting**
- **Component Organizer**
  - Small containers for screws and parts
- **Workspace Mat**
  - To prevent parts from rolling away

## Safety Equipment

- **Safety Glasses**
  - Required when using power tools
- **Work Gloves** (optional)
  - For handling sharp edges

## Software Tools

- **Arduino IDE** or compatible software
  - For programming the microcontroller
- **3D Modeling Software** (optional)
  - For modifications or custom parts

## Optional but Helpful

- **Heat Gun**
  - For heat-set inserts or cable management
- **Deburring Tool**
  - For cleaning 3D printed parts
- **Cable Ties**
  - For wire management
- **Label Maker**
  - For component identification

## Workspace Setup

Ensure you have:
- Adequate lighting
- Organized storage for small parts
- Clean, flat working surface
- Access to power outlets
- Proper ventilation for soldering
